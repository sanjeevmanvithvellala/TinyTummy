# 🥗 TinyTummy

### AI-Based Child Nutrition Monitoring System

---

## 📌 Overview

**TinyTummy** is an AI-powered application designed to monitor and improve children's eating habits.
It analyzes food images, estimates nutritional values, and provides smart recommendations to ensure a balanced diet.

---

## 🎯 Problem Statement

Many children today suffer from **imbalanced diets** due to:

* High consumption of junk food
* Lack of nutritional awareness
* Difficulty in tracking daily intake

This leads to poor health, weak immunity, and improper growth.

---

## 💡 Solution

TinyTummy provides a smart and automated way to:

* Track food intake using images
* Analyze nutrients using AI
* Generate health insights and reports
* Alert parents when diet is unhealthy

---

## 🚀 Features

* 📸 Food Image Recognition
* 🍎 Nutritional Analysis (Proteins, Carbs, Vitamins)
* 📊 Daily & Weekly Reports
* 🏆 Healthy Eating Reward System
* ⚠️ Smart Alerts for Poor Diet
* 👨‍👩‍👧 Parent Dashboard

---

## 🛠️ Tech Stack

**Frontend:**

* Streamlit

**Backend:**

* Python

**Database:**

* Firebase

**AI/ML:**

* Google Vision API / CNN Models
* Nutritional Prediction Algorithms

---

## 🔄 Workflow

1. User uploads a food image
2. AI detects food items
3. Nutritional values are calculated
4. Data is stored and tracked
5. Reports and health scores are generated
6. Alerts are sent if diet is unhealthy

---

## 📊 Output

* Nutrition charts
* Daily health score
* Weekly reports
* Personalized recommendations
